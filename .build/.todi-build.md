# TODO - A Simple TODO For Side Projects

### Profile

All Things Depends on Your Profile with One Properties

```json
{
    "name": "doit-user",
    "time": 270
}
```

The `time` Signifies Here that How Much Time you have in your Day



### Adding Project

```bash
$ doit add project 'projecta'
```

This will Create  a Series of Questions 

```bash
$ Time you Want to Spend in This Project [eg. 120]?
$ Currently Doning this Project [Y/N]: 
```

1. Name of Your Project
2. Time you Want to Spend in this Project
3. The max Projects you Can do At a time 

#### Data-Structure

```json
{
    "doit_version": "1.0.0",
    "doit_projecsts": [
        {
            id: 1
            name: "Project A",
            doing: true,
            time: 270     
        }
    ]
}
```



### Setting a Default Project 

A Default Project  is Set for Doing Boards

```bash
$ doit set default 
[1] projecta
[2] projectb
select The Default One : 
```

Setting Default Project is One From you want to do
