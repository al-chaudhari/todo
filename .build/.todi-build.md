# TODO

A Simple Todo With (Do, Delete, Defer, Delegate Logic)

### Profile

All Things Depends on Your Profile with One Properties

```json
{
    "name": "doit-user",
    "time": 270
}
```

The `time` Signifies Here that How Much Time you have in your Day



### Adding Project

```bash
$ doit add project 
```

This will Create  a Series of Questions 

```bash
$ Project Name ?
$ Time you Want to Spend in This Project [eg. 120]?
[1] ProjectA
[2] ProjectB
[3] projectC
$ Priority
```
